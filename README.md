# AWS Cloud Engineer Portfolio - React

A beautiful, animated React portfolio showcasing AWS Cloud Engineering skills with dark mode support.

## Features

- **Dark Mode Toggle**: Switch between light and dark themes with persistent storage
- **Smooth Animations**: Powered by Framer Motion
- **Responsive Design**: Works on all devices
- **Modern UI**: Clean, professional AWS-themed design
- **Interactive Elements**: Hover effects and scroll animations
- **Project Images**: Visual showcase of featured projects
- **Professional Experience**: Timeline-based experience section

## Setup

1. Install dependencies:
```bash
cd react-portfolio
npm install
```

2. Start development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Customization

- Update personal information in components
- Modify projects, skills, and experience data
- Add your own project images to `/public/images/`
- Adjust animations in component files
- Change colors and themes in CSS files

## New Components

- **Experience**: Professional timeline replacing certifications
- **DarkModeToggle**: Theme switcher with smooth transitions
- **ThemeContext**: Global state management for dark mode

## Technologies Used

- React 18
- Framer Motion (animations)
- CSS3 (styling with dark mode support)
- Context API (theme management)
- Responsive Grid Layout
